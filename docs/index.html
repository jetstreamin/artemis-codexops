How to Run

git clone https://github.com/jetstreamin/artemis-codexops.git
cd artemis-codexops
pip install -r requirements.txt
uvicorn cloudapi:app --host 0.0.0.0 --port 8080
# Open: http://localhost:8080/web/

Or deploy instantly to Fly.io or Heroku for your own cloud dashboard.
